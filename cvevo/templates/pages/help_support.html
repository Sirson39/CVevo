FAQs

Why is my ATS score low?
Usually missing keywords, weak summary, or formatting issues. Go to Analysis Results → Missing Keywords and apply suggestions.
What file types can I upload?
PDF and DOCX are recommended. If parsing fails, try exporting your resume again as a clean PDF.
How do I analyze against a new job description?
Analysis Results → Analyze New JD. Paste the JD and re-run scan.
Why is Export PDF not downloading?
Check if popups are blocked. Also try Export DOCX. If still failing, send a ticket with the browser name.
Can I create multiple resumes?
Yes. Use Resume Upload to store different versions and track scores per resume.
